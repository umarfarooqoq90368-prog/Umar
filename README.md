# Umar Farooq Portfolio

## Quick Start

### 1. Install Dependencies
```bash
npm install
```

### 2. Setup Email (Optional)
- Copy `.env.example` to `.env`
- Add your Gmail credentials
- Use **App Password** (not regular password) from Google Account settings

### 3. Start Server
```bash
npm start
```

### 4. Open Website
Open `index.html` in your browser or use a local server.

---

## Features
- ✅ Animated logo with your name
- ✅ Contact form saves to CSV file
- ✅ Email notifications (when configured)
